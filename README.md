# 🎧 airpods-finn

> Filtrert liste over de **billigste komplette AirPods** til salgs på FINN.no – under 500 kr.

🌐 **Live side:** [elvic-group.github.io/airpods-finn](https://elvic-group.github.io/airpods-finn/)

---

## 📋 Om prosjektet

Dette prosjektet samler og filtrerer AirPods-annonser fra [FINN.no](https://www.finn.no) og presenterer dem i en oversiktlig liste med direkte lenker til både annonse og selger-chat.

### ✅ Hva som er inkludert
- Komplette AirPods-sett (begge ørepropper + ladeetui)
- Kun annonser under **500 kr**
- Sortert fra billigst til dyrest
- Direkte lenke til produkt og selger-chat

### ❌ Hva som er filtrert bort
- Deksler, esker og emballasje
- Enkelt-ørepropper (kun venstre eller høyre)
- Defekte / ødelagte enheter
- Ladere og tilbehør
- Ikke-Apple merker (Samsung, Lenovo, JLab osv.)
- Falske / kopi-AirPods
- Reserverte eller solgte annonser

---

## 🔗 Lenker i listen

Hver annonse har to knapper:

| Knapp | Funksjon |
|-------|----------|
| 🔗 **Annonse** | Åpner produktsiden direkte på FINN.no |
| 💬 **Chat** | Åpner chat med selger direkte |
